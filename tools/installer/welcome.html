Welcome to Irrlicht

Irrlicht is a macOS menu bar application that monitors Claude Code sessions, providing real-time visual feedback through colored status indicators.

This installer will:

  • Install Irrlicht.app to /Applications
  • Install irrlicht-hook to /usr/local/bin
  • Configure Claude Code hooks automatically
  • Launch Irrlicht so it is immediately available in your menu bar

To uninstall, run:
sudo /usr/local/bin/irrlicht-uninstall.sh
