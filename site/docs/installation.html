Installation

Get Irrlicht running on your Mac in under two minutes via the package installer, or build everything from source.

Prerequisites

  • macOS 13 (Ventura) or later — Irrlicht uses SwiftUI APIs and LaunchAgent features that require Ventura as a minimum.
  • At least one supported AI coding agent installed and working:
    • Claude Code
    • OpenAI Codex CLI

If you plan to build from source, you will also need:

  • Go 1.21+ — the daemon is written in Go
  • Swift 5.9+ — the menu-bar app is a native SwiftUI application
  • Xcode Command Line Tools — install with xcode-select --install

Install via Package Installer (Recommended)

The fastest path. A signed .pkg installer handles everything for you.

  1. Download the installer

    Grab Irrlicht-0.2.0-mac-installer.pkg from the GitHub Releases page.

  2. Run the installer

    Double-click the .pkg file and follow the on-screen prompts. You may need to approve the installer in System Settings → Privacy & Security if Gatekeeper blocks it.

  3. What gets installed
    ComponentLocation
    irrlichd (daemon)/usr/local/bin/irrlichd
    Irrlicht.app (menu-bar app)/Applications/Irrlicht.app
    LaunchAgent plist~/Library/LaunchAgents/com.anthropic.irrlicht.daemon.plist

    The LaunchAgent ensures the daemon starts automatically every time you log in.

  4. Verify the installation 
    irrlichd --version

    You should see the version string printed to stdout (e.g. irrlicht 0.2.0).

Build from Source

Building from source gives you the full installer package, identical to the release download.

One-command build

git clone https://github.com/anthropics/irrlicht.git
cd irrlicht
./platforms/build-release.sh

The script produces a .pkg installer in the project root that you can double-click to install, exactly like the release download.

What the build script does

Under the hood, build-release.sh performs these steps in order:

  1. Syncs the web frontend — embeds the latest HTML/CSS/JS assets into the daemon binary.
  2. Cross-compiles the daemon — builds irrlichd for both arm64 (Apple Silicon) and amd64 (Intel).
  3. Creates a universal binary — merges both architectures with lipo so one binary runs natively on any Mac.
  4. Builds the Swift menu-bar app — compiles Irrlicht.app from the SwiftUI source.
  5. Packages everything into a .pkg installer — assembles the daemon, app, and LaunchAgent plist into a macOS installer package.

Manual build (individual components)

If you prefer to build each component separately:

Daemon only

# Build for the current architecture
go build -o irrlichd ./cmd/irrlichd

# Or cross-compile a universal binary manually
GOOS=darwin GOARCH=arm64 go build -o irrlichd-arm64 ./cmd/irrlichd
GOOS=darwin GOARCH=amd64 go build -o irrlichd-amd64 ./cmd/irrlichd
lipo -create -output irrlichd irrlichd-arm64 irrlichd-amd64

Menu-bar app only

cd platforms/macos/Irrlicht
xcodebuild -scheme Irrlicht -configuration Release build
Tip After a manual build you will need to copy irrlichd to /usr/local/bin/ and Irrlicht.app to /Applications/ yourself. You will also need to install the LaunchAgent plist manually — see the Uninstalling section for its expected path.

Post-Installation

Once installed (via either method), Irrlicht is ready to go.

  • The daemon starts automatically on login via the LaunchAgent. No manual launch required.
  • The menu-bar app appears in your menu bar as a sparkle icon. Click it to see the current state of all detected AI agent sessions.

Verify the daemon is running

curl http://127.0.0.1:7837/state

A successful response returns a JSON object with the current global state. If the daemon is not running, you will get a connection-refused error.

Check logs

Daemon logs are written to:

~/Library/Application Support/Irrlicht/logs/

Inspect these if the daemon fails to start or you encounter unexpected behaviour.

Note The daemon listens on 127.0.0.1:7837 by default. This port is not exposed to the network. See the Configuration page to change the bind address or port.

Uninstalling

To remove Irrlicht completely, delete the following files and directories:

# Stop the daemon first
launchctl bootout gui/$(id -u) ~/Library/LaunchAgents/com.anthropic.irrlicht.daemon.plist

# Remove installed files
rm /usr/local/bin/irrlichd
rm -rf /Applications/Irrlicht.app
rm ~/Library/LaunchAgents/com.anthropic.irrlicht.daemon.plist

# Remove application data and logs
rm -rf ~/Library/Application\ Support/Irrlicht/
Warning Removing the Application Support/Irrlicht/ directory deletes all local session history and configuration. Back up anything you want to keep before running the command above.
